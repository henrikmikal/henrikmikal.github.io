# Prosjektstruktur

Denne mappen inneholder ett prosjekt per mappe.

## Anbefalt oppsett per prosjekt

- `index.html` - egen prosjektside
- `dokumenter/no/` - norske dokumenter (PDF, DOCX, osv.)
- `dokumenter/en/` - engelske dokumenter
- `bilder/` - bilder og illustrasjoner
- `filer/` - andre filer (zip, datafiler, vedlegg)

Legg nye prosjekter i egne mapper med samme struktur.
